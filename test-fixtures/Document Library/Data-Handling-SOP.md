# Data Handling SOP

## Purpose

This document establishes procedures for handling sensitive data within the organization.

## Scope

Applies to all team members who access, process, or store organizational data.

## Procedures

### 1. Data Classification

All data must be classified according to sensitivity:
- **Public** - No restrictions
- **Internal** - Internal use only
- **Confidential** - Need-to-know basis
- **Restricted** - Highest protection

### 2. Storage Requirements

| Classification | Encryption | Access Control | Backup |
|---------------|-----------|----------------|--------|
| Public | Optional | None | Weekly |
| Internal | Required | Team-based | Daily |
| Confidential | Required | Individual | Daily |
| Restricted | Required | Approval-based | Real-time |

### 3. Retention Periods

- Operational data: 3 years
- Financial records: 7 years
- Legal documents: Permanent
