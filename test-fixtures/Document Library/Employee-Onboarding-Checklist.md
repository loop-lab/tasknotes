# Employee Onboarding Checklist

## Purpose

Standard checklist for onboarding new employees with security-relevant access and training requirements.

## Pre-Start (HR)

- [ ] Background check completed
- [ ] NDA signed
- [ ] Acceptable use policy signed
- [ ] Equipment ordered
- [ ] Badge/access card requested

## Day 1 (IT)

- [ ] Workstation configured
- [ ] Email account created
- [ ] VPN access provisioned
- [ ] MFA enrolled
- [ ] Password set and verified

## Week 1 (Security)

- [ ] Security awareness training completed
- [ ] Phishing simulation enrolled
- [ ] Data handling training completed
- [ ] Emergency procedures reviewed
- [ ] Incident reporting process explained

## Week 2 (Manager)

- [ ] Role-specific access granted
- [ ] Team introductions complete
- [ ] Tool-specific training scheduled
- [ ] First review date set
